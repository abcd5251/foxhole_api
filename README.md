## keyword Monitors (keyword monitoring)
Create a keyword monitor:
```bash
curl -X POST "https://foxhole.bot/api/v1/keywordMonitors" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR-API-KEY" \
  -d '{
    "name": "uxlink tweets",
    "keywords": ["@uxlink"]
  }'
```
